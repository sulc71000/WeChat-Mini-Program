# 心动日记 - 情侣空间小程序

> 一款面向情侣的微信小程序，提供动态分享、纪念日管理、照片相册等浪漫功能。

## 🎯 项目简介

「心动日记」是一款微信小程序，专为情侣设计，提供以下核心功能：

- 📅 **纪念日管理** - 记录重要日子，置顶功能，循环提醒
- 📷 **照片相册** - 瀑布流展示，情侣共享回忆
- 💬 **情侣动态** - 朋友圈式动态，支持点赞评论
- ❤️ **情侣空间** - 展示在一起天数，见证爱情

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| **微信小程序** | 原生框架 + 云开发 |
| **云函数** | Node.js 后端逻辑 |
| **云数据库** | 微信云开发 NoSQL |
| **云存储** | 图片音视频存储 |
| **Vant Weapp** | UI 组件库 |

## 🚀 核心功能实现

### 情侣绑定机制
- 双方扫码/邀请码绑定，共享情侣空间
- 数据隔离，保证隐私安全

### 纪念日系统
- 支持公历/农历日期
- 循环周期设置（每年/每月/每周）
- 置顶重要纪念日

### 瀑布流相册
- 响应式瀑布流布局
- 图片预览与缩放
- 本地缓存优化

## 🎨 界面设计

- **主题色**：浪漫红色系 (#FF6B6B)
- **设计语言**：渐变、圆角卡片、柔和动画
- **交互优化**：触摸反馈、骨架屏加载

## 📱 体验方式

1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目
3. **⚠️ 重要：替换 AppID**
   - 打开 `project.config.json`
   - 将 `YOUR_APPID` 替换为你的小程序 AppID
   - 在 [微信公众平台](https://mp.weixin.qq.com/) 获取 AppID
4. 开通云开发环境（免费）
5. 上传云函数：在 `cloudfunctions` 目录右键 → "上传并部署"
6. 编译预览
